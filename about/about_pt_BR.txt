A primeira versão desse plugin foi baseada no
código do Doitsu para o Plugin ACE do Sigil.
Esta é uma versão simplificada.

O plugin permite executar o ACE diretamente do
Editor. O relatório é aberto no navegador padrão.

O Menu 'Configurar' é baseado no KindleUnpack.

Obrigado ao Kovid Goyal pela ajuda na criação do
Menu 'Configurar' dentro do Editor.
